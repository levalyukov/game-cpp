# Miren (Miroro Engine) v1.0

Для создания хотя бы небольшой игры на базе SFML нужно разработать хотя бы базовые инструменты для дальнейшей создания видеоигры.

Цель — создать небольшой и минималистичный игровой движок с базовыми инструментами для комфортной работы и просто потренировать свои навыки в C++. Имеет на все инструменты тесты и документацию.

## Инструменты движка:

- Класс `EntityManager`: создание, удаление и манипуляция с сущностями. **[Читать подробнее...](./entity-manager.md)**
- Класс `UIManager`: создание, удаление и манипуляция UI-интерфейсами. **[Читать подробнее...](./ui-manager.md)**
- Класс `SceneManager`: создание, удаление и манипуляция с игровыми сценами. **[Читать подробнее...](./scene-manager.md)**
- Класс `EventManager`: создание, удание и манипуляция с игровыми событиями. **[Читать подробнее...](./event-manager.md)**
