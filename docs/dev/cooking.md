# Система готовки

Сердце игры - приготовление заказов для приходящий [клиентов](./orders.md#клиент).

Для того чтобы приступить к готовке: 
1. Нужно иметь хоть один открытый рецепт
2. Подойти к игровой плите и нажать ЛКМ по объекту
3. В открытом меню выберите рецепт
4. Приобретите все необходимые ингредиенты в [продуктовом магазине](./glocery-shop.md)
5. Нажмите кнопку "Готовить"
6. После истечения времени нажмите по игровой плите чтобы забрать готовое блюда

Рецепты можно получать в процессе игры.

## Класс `CookingManager`

Данный менеджер отвечает за хранение и добавление открытых рецептов, а также за процесс готовки.

### Методы

#### Добавить рецепт игроку
Добавляет рецепт, при условии что он есть в [главном классе](./items.md) `Items`.

```cpp
void addRecipe(const std::string recipe_name, uint16_t id);
```

#### Получение доступного рецепта
Возвращает структуру `CookRecipe`. Если указанного рецепта нет - вернет nullptr.
```cpp
Items::CookRecipe* getRecipe(const std::string recipe_name);
```

#### Начать приготовление
```cpp
void startCookProcess(const std::string recipe_name);
```

#### Процесс приготовление
Вызывается каждый кадр
```cpp
void cookingProcess(const float delta);
```

#### Получить флаг приготовления в данный момент
```cpp
bool getCookingFlag() const;
```

#### Получить флаг приготовления
```cpp
bool getCookedFlag() const;
```

#### Получить название готовящего рецепта
```cpp
std::string getCookingRecipeName() const;
```

#### Сбрасывает флаг приготовленности блюда
```cpp
void resetCookedFlag();
```
