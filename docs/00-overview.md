# Обзор

- **Название**: Вкусный Хлеб (Delicious Soup)
- **Жанр**: 2D pixel top-down simulator
- **Платформы**: Windows, Linux, macOS
- **Языки**: Русский, Английский
- **Технологии**: C++20, MSVC, SFML, Catch2, ImGui 

### Управление

| Клавиша(-и)      | Действие                  | 
|----------------- | ------------------------- |
| **WASD**         | Передвижение              |
| **WASD + SHIFT** | Ускоренное передвижение   |
| **ЛКМ**          | Взаимодействие с объектом |
| **ПКМ**          | Взаимодействие с клиентом |
| **ESC**          | Меню паузы                |


### Используемые библиотеки:
- **[SFML v2.6.2](https://github.com/SFML/SFML)** - Основная библиотека для взаимодействия с окном, выводом, звуком, спрайтами и т.д.
- **[Catch2 v3.5.0](https://github.com/catchorg/catch2)** - Модульные тесты
- **[imGui](https://github.com/ocornut/imgui)** - GUI для отладки
